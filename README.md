# 시간표 자동 배정 웹앱

참가자가 가능 시간을 제출하면, 조건에 맞춰 자동으로 근무 시간표를 짜주는 웹앱이에요.

## 들어간 조건
- 인원 균등 배치
- 자리가 안 차면 빈칸 허용 (장소 우선순위 순으로 채움)
- 장소별 운영시간 다르게 설정 가능
- 한 장소에 들어가면 **최소 연속 시간 이상** 근무 (토막 근무 금지)
- 다른 장소로 이동 가능, 단 거기서도 연속 최소시간 이상
- 하루 최대 근무시간 제한

---

## 처음 한 번만: 설치

1. 컴퓨터에 **Python**이 있어야 해요. 없으면 https://www.python.org 에서 설치 (설치할 때 "Add Python to PATH" 체크).
2. 이 폴더(`schedule_app`)에서 터미널(명령 프롬프트)을 열고:

```
pip install -r requirements.txt
```

> Windows에서 터미널 여는 법: 폴더 주소창에 `cmd` 입력 후 Enter.

## 실행

같은 폴더에서:

```
python app.py
```

그러면 이런 메시지가 나와요:

```
시간표 배정 서버 시작!  ->  http://localhost:8000
```

브라우저에서 **http://localhost:8000** 접속.

## 사용 순서

1. **주최자(나)**: 첫 화면에서 제목·요일·운영시간·장소(정원/우선순위/운영시간)·최소연속/최대시간을 입력하고 **링크 만들기**.
   - **참가자 링크**와 **관리자 링크** 2개가 나와요.
2. **참가자**: 참가자 링크를 받아서 이름 입력 + 가능 시간 드래그 → **제출하기**.
3. **주최자**: 관리자 링크에 들어가 제출 현황 확인 → **배정 실행** → 결과표 확인.

## 같은 와이파이의 다른 사람도 접속하려면
- 서버 실행 후, 내 컴퓨터의 IP(예: `192.168.0.5`)를 확인해서
- 참가자에게 `http://192.168.0.5:8000/e/...` 형태로 링크를 주면 돼요.
- (외부 인터넷 공유는 별도 배포 필요 — Render 등. 원하면 도와줄게요.)

## 파일 구성
- `app.py` — 웹 서버 (FastAPI)
- `solver.py` — 배정 엔진 (OR-Tools CP-SAT)
- `static/` — 화면 3개 (주최자/참가자/관리자)
- `schedule.db` — 데이터 저장 (자동 생성)

## 설정 바꾸기 팁
- 배정 성향은 `solver.py`의 가중치로 조절: `cover`(빈칸 최소), `balance`(균등), `frag`(연속 선호).
- 배정 계산 시간은 `app.py`의 `time_limit`(기본 30초).
